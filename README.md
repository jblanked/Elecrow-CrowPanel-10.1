## Elecrow CrowPanel 10.1"

This repository contains C and MicroPython libraries and examples for Elecrow's new CrowPanel 10.1" development board.

The device features:
- 10-inch screen powered by the RISC-V-based ESP32-P4 SoC
- Combined with 16MB of Flash and 32MB of PSRAM
- Equipped with dual USB-C ports (UART adapter/USB 2.0 Device), 40-pin GPIO, multiple Crowtail interfaces (supporting I2C/UART protocols), and a replaceable wireless module slot (supporting Thread, Zigbee, Halow, etc.)
- Onboard ESP32-C6-MINI-1 module supports 2.4GHz Wi-Fi 6 and Bluetooth 5.3
